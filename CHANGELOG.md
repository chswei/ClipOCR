# ClipOCR Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Extract text from selected Windows screen areas with Windows Screen Snip and local Tesseract OCR.
- Copy recognized text to the clipboard.
- Support mixed Tesseract language codes such as `chi_tra+eng`.
- Add configurable Tesseract page segmentation mode.
- Add Chinese punctuation cleanup for Traditional Chinese and mixed Traditional Chinese-English OCR.
